
کلاس های concrete 
در برنامه نویسی شئ گرا کلاس هایی هستند که هیچ مفهوم اتزاعی
 (متد abstract یا عضو virtual)
 در آنها وجود ندارد و تمام متد های آنها دارای پیاده سازی هستند و در نتیجه می توان از آنها شئ ساخت.

 A concrete class is a class that allows creating an instance or an object using the new keyword.
 A concrete method is a method that can be called directly.


Creational patterns
These patterns provide various object creation mechanisms, which increase flexibility and reuse of existing code.
این الگوها مکانیسم های مختلف ایجاد شی را ارائه می دهند که انعطاف پذیری و استفاده مجدد از کد موجود را افزایش می دهد

Structural patterns
These patterns explain how to assemble objects and classes into larger structures 
while keeping these structures flexible and efficient.
این الگوها نحوه جمع آوری اشیاء و کلاس ها را در ساختارهای بزرگتر توضیح می دهند و در عین حال این ساختارها را انعطاف پذیر و کارآمد نگه می دارند

Behavioral patterns
These patterns are concerned with algorithms and the assignment of responsibilities between objects.
این الگوها با الگوریتم ها و تخصیص مسئولیت ها بین اشیا سروکار دارند
